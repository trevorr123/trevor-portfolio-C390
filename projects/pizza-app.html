Overview

Trevor’s Pizza Co. is a responsive web application that showcases a dynamic pizza menu with images, descriptions, and pricing. Users can browse available pizzas, search by name or ingredients, and interact with the ordering interface. The app includes conditional rendering to display store status (open or closed) and adjusts the call-to-action accordingly. This project demonstrates my ability to build interactive, data-driven user interfaces, implement real-time search functionality, and apply conditional logic within a modern web application.

What I Did

  • Designed and built a responsive pizza menu using reusable card components
  • Structured menu data and dynamically rendered items from organised data arrays
  • Implemented real-time search functionality to filter pizzas by name or ingredients
  • Applied conditional rendering to display store open/closed status
