Overview

This Card App is a full-stack web application that allows users to manage a collection of cards using basic CRUD functionality. Users can create new cards, view all existing cards in a clean grid layout, update card details, and delete cards when needed. The frontend provides an interactive interface with Edit and Delete actions, while the backend handles API requests and communicates with the database to store and retrieve card data. This project demonstrates my understanding of full-stack development, including building user interfaces, implementing RESTful APIs, and connecting the application to a database for persistent data management.

What I Did

  • Designed and built a responsive card-based user interface to display items clearly
  • Implemented full CRUD functionality (Create, Read, Update, Delete)
  • Developed RESTful API routes to handle frontend requests
  • Connected the backend to a database for persistent data storage
  • Integrated frontend and backend to enable dynamic data updates
