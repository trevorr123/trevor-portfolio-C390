Overview

The Travel List App is a responsive web application that helps users organise and track items they need to pack for a trip. Users can add items with quantities, mark them as packed, sort the list, and remove individual items or clear the entire list. The app dynamically updates the interface and calculates packing progress in real time, providing a simple and intuitive user experience. This project demonstrates my understanding of state management, dynamic rendering, and interactive UI development.

What I Did

  • Designed and built a responsive packing list interface
  • Implemented functionality to add items with quantity selection
  • Enabled users to mark items as packed using checkboxes
  • Implemented item deletion and “clear list” functionality
  • Added sorting options to organise the list dynamically
  • Displayed real-time packing progress and percentage completion
