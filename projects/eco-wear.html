Overview

EcoWear is a UI kit created in Figma focusing on a clean visual system—typography, colours, buttons, cards, forms, and reusable components. The design aims for a modern, minimal look that supports a smooth shopping experience.

What I Did

  • Designed typography scale and colour palette
  • Built core UI components (buttons, cards, inputs, nav)
  • Created form and data elements for consistency
  • Maintained spacing, alignment, and responsive layout planning
